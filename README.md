# Gigatron TTL Simulator

## Usage

1. Setup Rust (see https://www.rust-lang.org/tools/install):

    $ curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

2. Compile and run:

    $ cargo run
